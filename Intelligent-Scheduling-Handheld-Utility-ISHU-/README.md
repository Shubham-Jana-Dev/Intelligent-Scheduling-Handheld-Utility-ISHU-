# Intelligent-Scheduling-Handheld-Utility-ISHU-
I.S.H.U. (Intelligent Scheduling Handheld Utility): A custom, time-aware AI assistant housed in a mini, portable Raspberry Pi robot. Features include persistent routine management (time slot checking, midnight wrap-around), voice control (gTTS/Speech Recognition), weather, and personalization. Perfect for on-the-go utility!
